## ✨ Overview

Haiku is a lightweight and efficient tool designed to streamline the creation of React stories for Storybook. By leveraging Abstract Syntax Tree (AST) parsing via Babel, Haiku simplifies the process of generating and managing your Storybook stories, allowing developers to focus more on component development and less on boilerplate.

## 🚀 Features

- Automated Story Generation: Automatically generates basic Storybook stories from your React components.
- AST-Powered Parsing: Utilizes Babel Parser for robust and accurate code analysis.
- Simplified Workflow: Reduces the manual effort required to set up and maintain Storybook stories.
- Extensible Design: Built with extensibility in mind, allowing for future enhancements and custom parsers.
